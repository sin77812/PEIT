# 네이버 검색 최적화 가이드

## 현재 상태 확인

✅ 네이버 사이트 인증: 완료 (`naver-site-verification` 메타 태그 설정됨)
✅ robots.txt: 설정 완료 (네이버 검색로봇 Yeti 허용)
✅ sitemap.xml: 생성 완료 (30개 페이지 포함)
✅ 구조화된 데이터: Schema.org WebApplication 설정됨

## 네이버 검색이 안 되는 주요 이유

1. **인덱싱 시간**: 네이버가 사이트를 크롤링하고 인덱싱하는데 **1-2주** 정도 소요됩니다.
2. **사이트맵 미제출**: 네이버 웹마스터 도구에 사이트맵이 제출되지 않았을 수 있습니다.
3. **URL 미제출**: 주요 페이지 URL이 제출되지 않았을 수 있습니다.
4. **콘텐츠 품질**: 네이버는 품질이 낮은 콘텐츠를 인덱싱하지 않을 수 있습니다.

## 해결 방법 (단계별)

### 1단계: 네이버 웹마스터 도구 확인

1. **네이버 웹마스터 도구 접속**
   - https://searchadvisor.naver.com
   - 로그인 후 사이트 선택

2. **사이트맵 제출 확인**
   - 왼쪽 메뉴: **"요청"** > **"사이트맵 제출"**
   - 사이트맵 URL 입력: `https://www.peit24.co.kr/sitemap.xml`
   - **"확인"** 버튼 클릭
   - 제출 상태 확인 (보통 몇 시간~하루 소요)

3. **URL 제출 (중요!)**
   - 왼쪽 메뉴: **"요청"** > **"URL 제출"**
   - 다음 URL들을 하나씩 제출:
     ```
     https://www.peit24.co.kr/
     https://www.peit24.co.kr/test
     https://www.peit24.co.kr/types
     https://www.peit24.co.kr/about
     ```
   - 각 URL 제출 후 "제출 완료" 메시지 확인

4. **robots.txt 수집 요청**
   - 왼쪽 메뉴: **"요청"** > **"robots.txt"**
   - **"수집요청"** 버튼 클릭
   - 최신 robots.txt 수집 확인

### 2단계: 검색 최적화 확인

1. **robots.txt 검증**
   - 왼쪽 메뉴: **"검증"** > **"robots.txt 검증"**
   - URL 입력: `https://www.peit24.co.kr/`
   - **"확인"** 버튼 클릭
   - "수집 가능" 상태 확인

2. **사이트 상태 확인**
   - 왼쪽 메뉴: **"검증"** > **"사이트 상태"**
   - 크롤링 오류 확인
   - 인덱싱 상태 확인

### 3단계: 콘텐츠 최적화

네이버 검색을 위해 각 페이지에 적절한 제목과 설명이 필요합니다.

### 4단계: 대기 및 모니터링

- **1-2주 대기**: 네이버가 사이트를 크롤링하고 인덱싱하는데 시간이 필요합니다.
- **정기 확인**: 네이버 웹마스터 도구에서 인덱싱 상태 확인
- **검색 테스트**: 네이버에서 `site:peit24.co.kr` 검색으로 인덱싱 확인

## 즉시 확인할 수 있는 방법

네이버 검색창에서 다음을 검색해보세요:
```
site:peit24.co.kr
```

결과가 나오면 인덱싱이 시작된 것입니다.

## 추가 권장사항

1. **정기적인 콘텐츠 업데이트**: 네이버는 활발히 업데이트되는 사이트를 선호합니다.
2. **모바일 최적화**: 모바일 친화적인 사이트가 검색 순위에 유리합니다.
3. **페이지 속도**: 빠른 로딩 속도가 중요합니다.
4. **백링크**: 다른 사이트에서의 링크가 있으면 인덱싱이 빨라집니다.
