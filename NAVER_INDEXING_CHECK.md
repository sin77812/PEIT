# 네이버 인덱싱 상태 확인 방법

## 네이버 웹마스터 도구에서 확인하는 방법

### 1. 검색 현황 (인덱싱된 페이지 수 확인)

**경로:**
- 네이버 웹마스터 도구 접속: https://searchadvisor.naver.com
- 왼쪽 메뉴: **"검색 현황"** > **"검색 노출 현황"**

**확인 내용:**
- 인덱싱된 페이지 수
- 검색 노출 횟수
- 클릭 횟수
- 노출 키워드

### 2. 사이트맵 제출 현황

**경로:**
- 왼쪽 메뉴: **"요청"** > **"사이트맵 제출"**

**확인 내용:**
- 제출된 사이트맵 URL
- 사이트맵 처리 상태 (대기중/처리완료/오류)
- 사이트맵에 포함된 URL 수
- 인덱싱된 URL 수

### 3. URL 제출 현황

**경로:**
- 왼쪽 메뉴: **"요청"** > **"URL 제출"**

**확인 내용:**
- 제출한 URL 목록
- 각 URL의 처리 상태
- 인덱싱 완료 여부

### 4. 사이트 상태 (크롤링 오류 확인)

**경로:**
- 왼쪽 메뉴: **"검증"** > **"사이트 상태"**

**확인 내용:**
- 크롤링 오류 목록
- 접근 불가 페이지
- 리다이렉트 오류
- 404 오류

### 5. 수집 요청 현황

**경로:**
- 왼쪽 메뉴: **"요청"** > **"수집 요청"**

**확인 내용:**
- 수집 요청한 URL 목록
- 수집 상태 (대기중/수집완료/오류)
- 수집 완료 시간

## 네이버 검색에서 직접 확인하는 방법

### 1. site: 명령어 사용

네이버 검색창에서 다음을 검색:
```
site:peit24.co.kr
```

**결과 해석:**
- 결과가 나오면: 인덱싱이 시작된 것입니다
- 결과가 없으면: 아직 인덱싱되지 않았거나 크롤링 중입니다

### 2. 특정 페이지 검색

```
site:peit24.co.kr 정치성향테스트
```

또는

```
site:peit24.co.kr PEIT
```

## 인덱싱이 안 되는 경우 확인사항

### 1. robots.txt 확인
- **경로:** "요청" > "robots.txt"
- **확인:** 네이버 검색로봇(Yeti)이 차단되지 않았는지 확인
- **해결:** "수집요청" 버튼 클릭하여 최신 robots.txt 수집

### 2. 사이트맵 제출 확인
- **경로:** "요청" > "사이트맵 제출"
- **확인:** 사이트맵이 제출되었는지, 처리 상태는 어떤지 확인
- **해결:** 사이트맵 URL 제출: `https://www.peit24.co.kr/sitemap.xml`

### 3. URL 제출 확인
- **경로:** "요청" > "URL 제출"
- **확인:** 주요 페이지 URL이 제출되었는지 확인
- **해결:** 메인 페이지 URL 제출: `https://www.peit24.co.kr/`

### 4. 크롤링 오류 확인
- **경로:** "검증" > "사이트 상태"
- **확인:** 크롤링 오류가 있는지 확인
- **해결:** 오류가 있으면 해당 페이지 수정

## 인덱싱 시간

- **일반적으로:** 1-2주 소요
- **빠른 경우:** 3-7일
- **느린 경우:** 2-4주

## 인덱싱을 빠르게 하는 방법

1. **사이트맵 제출** (가장 중요!)
2. **URL 직접 제출** (주요 페이지)
3. **정기적인 콘텐츠 업데이트**
4. **백링크 확보** (다른 사이트에서 링크)
5. **소셜 미디어 공유** (네이버 블로그, 카페 등)
